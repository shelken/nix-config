# lessons

- 2026-04-01: 创建项目级 skill 前，先确认 skill 的归属层级。当前项目的 skill 应放在项目根目录
  `.agents/skills/` 下，不要误放到 `home/base/gui/dev/ai/skills/` 这类用户环境配置目录。
- 2026-04-05: 用户要求保留被禁用配置项的注释和问题背景，不要直接删除；配置 workaround 需要附上相关 issue 和现象说明。
- 2026-04-06: `lessons.md`
  只记录可复用的决策规则，不记录单次任务结论。规则应能指导后续同类问题的取舍，例如“共享输入走统一入口注入，避免多处重复定义”，而不是“这次把某个字段放到某处”。
- 2026-04-06: 外部资源替换本地打包逻辑前，先核对调用方依赖的文件语义是否保持一致，尤其是可执行位、目录结构、文件名和运行时路径。
- 2026-04-06: 迁移配置到上游新插件或新接口时，不能沿用旧配置字段名；必须逐项对照官方示例确认匹配键、参数名和调用约定。
